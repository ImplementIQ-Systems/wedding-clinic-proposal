# Wedding Clinic — Proposta de Website

Proposta de redesign do website para a **Wedding Clinic**, loja de vestidos de noiva exclusivos em Seixal, Portugal.

Desenvolvido por [ImplementIQ Systems](https://implementiq-systems.github.io/ImplementIQ/)

---

## Versões disponíveis

| Ficheiro | Estilo | Descrição |
|---|---|---|
| `index-dark.html` | Dark / Luxo | Fundo escuro, tons dourados, posicionamento premium |
| `index-light.html` | Light / Elegante | Fundo claro, tons neutros, estética moderna e limpa |

---

## O que está incluído nesta proposta

- Homepage com hero, destaque de vestidos e secção de marcação
- Catálogo de vestidos com filtros por coleção (Signature / Flash)
- Efeito elegante de zoom ao passar o rato nos vestidos
- Botão WhatsApp flutuante
- Secção de marcação de visitas
- Design responsivo (mobile-friendly)
- Rodapé com contactos e redes sociais

---

## Roadmap do projecto completo

### Basic
- Site completo migrado do Wix
- Catálogo de vestidos gerível
- Sistema de marcações integrado
- Botão WhatsApp flutuante
- Blog

### Premium (adicional ao Basic)
- Dashboard de gestão de conteúdo
- Publicação simultânea em Instagram, Facebook e TikTok
- Rascunhos automáticos para retomar quando conveniente

---

## Stack prevista

- Frontend: HTML / CSS / JS (estático, rápido, sem dependências)
- Hosting: GitHub Pages ou VPS Hetzner
- Marcações: Calendly integrado ou solução custom
- Dashboard: Integração com Buffer API
